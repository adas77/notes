  - `Java 17`

  - `...`

  - `POSTGRESQL`
- Server

- Run
  - `Docker` & `docker-compose`

## Konfiguracja
File: `.env`
## Run `docker-compose`
With `Makefile`:
- `make build` budowanie kontenerów
- `make up` uruchomienie kontenerów
- `make db` połączenie z bazą danych `POSTGRES` przez konsolę
